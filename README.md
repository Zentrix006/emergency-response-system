# 🧠 Problem Statement

```bash
$ problem
Hotels lack a unified system for:
❌ Immediate guest emergency reporting
❌ Real-time staff coordination
❌ Visual floor-based alert tracking
❌ Centralized incident management
```

# 💡 Solution

```bash
$ solution
✔ Guests scan QR → Report issue instantly
✔ Alerts pushed to Firebase (real-time)
✔ Staff dashboards update instantly
✔ Floor maps highlight exact room
✔ Departments act without delay
```

# ⚙️ System Architecture

```bash
Guest (QR Scan)
      ↓
Report Page (/report?room=301)
      ↓
Firebase Firestore 🔥
      ↓
Staff Dashboard (/staff)
      ↓
Admin Panel (/admin)
```

# 🧪 How It Works

```bash
$ flow
1. Guest scans QR in room
2. Opens /report?room=XXX
3. Selects issue (Fire / Medical / Security)
4. Alert stored in Firebase
5. Staff dashboard updates instantly
6. Room highlighted on floor map
7. Staff responds immediately
```
